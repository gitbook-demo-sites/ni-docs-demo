# Translation experience

This demo includes French and Chinese translation runs for the primary docs center content. The goal is to show how NI can support regional engineering teams without forking the English source.

## Demo languages

| Language | Use case | Notes |
| --- | --- | --- |
| French | European field engineering and support teams | Localized navigation and documentation body |
| Chinese | APAC engineering, manufacturing, and support teams | Localized navigation and documentation body |

{% hint style="info" icon="rotate" %}
In production, translations should be generated from the canonical English space after reviewed updates are merged.
{% endhint %}
