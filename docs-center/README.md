## What changes in GitBook

{% columns %}
{% column %}
#### One catalog, many entry points

The current NI portal is organized around software, all documentation, and product families. This demo keeps that mental model but gives each audience a focused space instead of one long product index.
{% endcolumn %}

{% column %}
#### Search across products

GitBook search and AI assistant can answer across manuals, API docs, release notes, support pages, and training content without sending readers through separate surfaces.
{% endcolumn %}

{% column %}
#### Global by default

French and Chinese translations are generated from the same English source, so local docs inherit updates without copying pages by hand.
{% endcolumn %}
{% endcolumns %}
