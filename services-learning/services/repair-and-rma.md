# Repair and RMA

Repair workflows should be easy to find from both hardware pages and support pages.

{% stepper %}
{% step %}
Confirm the product model, serial number, and observed failure.
{% endstep %}

{% step %}
Check troubleshooting guidance and known issues for the product version.
{% endstep %}

{% step %}
If the issue persists, open a service request and select repair or RMA.
{% endstep %}

{% step %}
Package and ship the device following NI service instructions.
{% endstep %}
{% endstepper %}
