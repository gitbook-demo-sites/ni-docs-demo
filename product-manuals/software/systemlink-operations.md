# SystemLink operations

SystemLink software provides software deployment, systems configuration, test monitoring, and data management tools for test and measurement systems.

## Operating model

<table data-view="cards"><thead><tr><th></th><th></th></tr></thead><tbody>
<tr><td><i class="fa-boxes-stacked" style="color:$primary;">:boxes-stacked:</i> <strong>Assets</strong></td><td>Track systems, hardware assets, calibration state, and availability.</td></tr>
<tr><td><i class="fa-cloud-arrow-up" style="color:$primary;">:cloud-arrow-up:</i> <strong>Deployment</strong></td><td>Push software packages and updates to managed systems.</td></tr>
<tr><td><i class="fa-chart-line" style="color:$primary;">:chart-line:</i> <strong>Monitoring</strong></td><td>Observe health, test execution, and utilization across labs and production.</td></tr>
<tr><td><i class="fa-database" style="color:$primary;">:database:</i> <strong>Data</strong></td><td>Aggregate test data for analysis, compliance, and reporting.</td></tr>
</tbody></table>
