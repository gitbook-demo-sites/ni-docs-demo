# Using the docs center

Start with the job you need to complete, then narrow by product family.

{% stepper %}
{% step %}
Search for the product name or family: LabVIEW, CompactDAQ, PXI, SystemLink, FlexLogger, or Package Manager.
{% endstep %}

{% step %}
Use product filters to separate user manuals, programming references, specifications, release notes, and getting started guides.
{% endstep %}

{% step %}
Open the latest version unless your deployed system is pinned to a specific product release.
{% endstep %}

{% step %}
Download a PDF or use offline help when you need bench-side access without a network connection.
{% endstep %}
{% endstepper %}

{% hint style="warning" icon="triangle-exclamation" %}
For regulated test systems, record the exact product version and document revision you used during validation.
{% endhint %}
