# Calibration

Calibration documentation connects hardware specifications, service intervals, certificates, and account workflows.

## Calibration record

| Field | Why it matters |
| --- | --- |
| Product and serial number | Identifies the instrument |
| Calibration date | Establishes validity period |
| Operator or provider | Supports traceability |
| Certificate | Supports audits and compliance |

{% hint style="warning" icon="calendar" %}
Schedule calibration before critical validation windows to avoid downtime.
{% endhint %}
