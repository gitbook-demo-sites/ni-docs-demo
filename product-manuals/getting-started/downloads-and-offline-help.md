# Downloads and offline help

NI users often work in labs, production environments, or secure networks where internet access is limited. The docs experience should make download and offline paths obvious.

## Options

| Need | Path |
| --- | --- |
| Install or update software | NI Package Manager or the software downloads portal |
| Save a manual for the bench | Download PDF from the product manual |
| Search without internet | Use NI Offline Help Viewer |
| Validate a deployed system | Archive the product version, manual revision, and release notes |

{% hint style="info" icon="file-pdf" %}
PDF downloads are still valuable for bench-side use, but GitBook can become the canonical source for search, AI answers, review history, and translations.
{% endhint %}
