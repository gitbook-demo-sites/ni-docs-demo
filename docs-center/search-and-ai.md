# Search and AI assistant

NI's documentation spans manuals, PDF downloads, setup guides, software help, API references, and support workflows. A GitBook demo should show search as a product capability, not just a text box.

## Indexed sources

{% columns %}
{% column %}
#### Structured docs

Manuals, getting started guides, specifications, product references, and release notes should live as GitBook pages with version history and reviews.
{% endcolumn %}

{% column %}
#### External content

Community posts, code examples, training catalogs, and support articles can be added as indexed external sources so the assistant can recommend them without forcing a migration on day one.
