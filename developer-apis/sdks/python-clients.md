# Python clients

The `nisystemlink-clients` package lets Python applications interact with a SystemLink Server or SystemLink Enterprise installation.

## Install

{% tabs %}
{% tab title="pip" %}
```bash
python -m pip install nisystemlink-clients
```
{% endtab %}

{% tab title="uv" %}
```bash
uv add nisystemlink-clients
```
{% endtab %}
{% endtabs %}
