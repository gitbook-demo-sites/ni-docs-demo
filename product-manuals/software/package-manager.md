# Package Manager

NI Package Manager installs, updates, repairs, and removes NI software. In a GitBook version of the docs, Package Manager becomes a cross-linking hub between software manuals, driver guidance, and release notes.

## Common flows

{% tabs %}
{% tab title="Install" %}
Use Package Manager to install NI applications, drivers, and dependencies from approved feeds.
{% endtab %}

{% tab title="Update" %}
Review release notes first, then update packages in a controlled maintenance window.
{% endtab %}

{% tab title="Repair" %}
Use repair when a local install is missing files or dependencies but the version should remain unchanged.
{% endtab %}

{% tab title="Remove" %}
Remove unused products and confirm downstream projects are no longer referencing those drivers.
{% endtab %}
{% endtabs %}
