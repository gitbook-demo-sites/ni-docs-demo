# Asset management

Asset APIs help teams understand where hardware is located, whether it is connected, and what calibration or metadata is associated with it.

## Example resource model

| Resource | Description |
| --- | --- |
| Asset | A managed item in a test system |
| Location | Where the asset is assigned or detected |
| Presence | Whether the asset is currently observed by a system |
| Calibration | Last calibration metadata and operator details |
