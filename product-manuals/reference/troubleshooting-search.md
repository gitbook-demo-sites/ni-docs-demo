# Troubleshooting search results

Large product catalogs can overwhelm readers when product names, driver names, and system names overlap.

## Better filters

| Filter | Example |
| --- | --- |
| Product family | LabVIEW, SystemLink, CompactDAQ, PXI |
| Document type | User manual, API reference, specification, release notes |
| Product version | Current release, long-term support, archived |
| Task | Install, configure, program, troubleshoot, calibrate |

{% hint style="info" icon="magnifying-glass" %}
In GitBook, search analytics can identify failed queries and help the docs team add redirects, synonyms, or missing anchor pages.
{% endhint %}
